# YTSearcher

Фреймворк:
`React`

UI фреймворк:
`Ant Design`

БД:
`Firebase`

API:
`YouTube API`

Вертска:
`Для компьтерных разрешений`

### Установка
```
npx create-react-app
npm install react-router-dom
npm install antd
npm install --save firebase
npm install --save youtube-api
```
